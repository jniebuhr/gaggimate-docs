This project upgrades a Gaggia espresso machine with smart controls to improve your coffee-making experience. By adding a display and custom electronics, you can monitor and control the machine more easily.

## Feature list and comparison

| Feature                         | GaggiMate Standard                           | GaggiMate Pro |
|---------------------------------|----------------------------------------------| ------------- |
| Minimalistic Touch UI | <i className="fa fa-check text-green-600" /> | <i className="fa fa-check text-green-600" /> |
| PID temperature control         | <i className="fa fa-check text-green-600" /> | <i className="fa fa-check text-green-600" /> |
| Simple brew profiles            | <i className="fa fa-check text-green-600" /> | <i className="fa fa-check text-green-600" /> |
| Profiles with pressure / flow control  | <i className="fa fa-times text-red-600" />   | <i className="fa fa-check text-green-600" /> |
| Exit conditions for profile phases | <i className="fa fa-times text-red-600" />   | <i className="fa fa-check text-green-600" /> |
| Phase transitions  | <i className="fa fa-times text-red-600" />   | <i className="fa fa-check text-green-600" /> |
| Shot diagrams                       | <i className="fa fa-check text-yellow-600" /> **\*** | <i className="fa fa-check text-green-600" /> |
| Pro Steam (Assists steaming with the pump)     | <i className="fa fa-check text-yellow-600" /> **\*\*** | <i className="fa fa-check text-green-600" /> |
| Brew by weight | <i className="fa fa-check text-green-600" /> | <i className="fa fa-check text-green-600" /> |
| Grind by weight (Relay / Tasmota) | <i className="fa fa-check text-green-600" /> | <i className="fa fa-check text-green-600" /> |
| Homekit integration | <i className="fa fa-check text-green-600" /> | <i className="fa fa-check text-green-600" /> |
| Home Assistant integration | <i className="fa fa-check text-green-600" /> | <i className="fa fa-check text-green-600" /> |
| Hot water function | <i className="fa fa-check text-green-600" /> | <i className="fa fa-check text-green-600" /> |
| Web UI | <i className="fa fa-check text-green-600" /> | <i className="fa fa-check text-green-600" /> |
| OTA updates | <i className="fa fa-check text-green-600" /> | <i className="fa fa-check text-green-600" /> |
| Bluetooth scales | <i className="fa fa-check text-green-600" /> | <i className="fa fa-check text-green-600" /> |
| Headless mode | <i className="fa fa-check text-green-600" /> | <i className="fa fa-check text-green-600" /> |

 **\*** Shot diagrams will only include weight and temperature for Standard
 **\*\*** Steam Assist is supported but won't be as smooth as on Pro, future pressure based steaming options are also not supported
