If you don't feel like ordering a PCB or kit and are more comfortable connecting up different components you can go this route too. Below you'll find all the components you can use to build up your own GaggiMate Pro Lego.

* ESP-S3 DevKit - N8R2 [(AliExpress)](https://s.click.aliexpress.com/e/_EuVDj60)
* RoboDyn AC Dimmer [(AliExpress)](https://s.click.aliexpress.com/e/_EHUTAaU) (Dimmer 4A-400V)
* 1-Channel SSR [(AliExpress)](https://s.click.aliexpress.com/e/_EviCKOc) (1-Channel high 5V)
* TXS0102 Logic Level Converter [(AliExpress)](https://a.aliexpress.com/_EQCtKUC)
* ADS1115 ADC [(AliExpress)](https://s.click.aliexpress.com/e/_Ew78C8Y)
* MAX31855 Temperature sensor board [(AliExpress)](https://s.click.aliexpress.com/e/_EuyTWd6)
* 10K and 100K Resistors to form a  [(AliExpress)](https://s.click.aliexpress.com/e/_EzZPgrq)

* Electrical tape for insulating unused connections
* Multimeter

* Optional: Zipties
* Optional: Small flathead screwdriver to remove connections
